    Block DX Installation¶

    Block DX is a decentralized exchange dApp powered by the Blocknet Protocol. Installing the Block DX is a simple process. Below are step-by-step instructions for how to install on each operating system (OS). If you already have Block DX installed and are updating, see the updating guide. For a complete guide on setting up Block DX, see the Block DX setup guide.

    Install Block DX¶

    Windows

    1. Download the latest Block DX release. There are 3 Windows options:

      1. BLOCK-DX-x.x.x-win.exe (recommended)
      2. BLOCK-DX-x.x.x-win-x64.zip
      3. BLOCK-DX-x.x.x-win-ia32.zip

    2. Before continuing, it is recommended to verify your download

    3. Continue to the instructions below for the download you selected:
    Using the .exe download (option a)
    1. Find the downloaded file. The default location is in Downloads.
    2. Double-click the file to begin the installation process.

    3. Read the License Agreement and select Accept to acknowledge.

    Installation Complete!

    Continue to related guides:

    • BlockDX Configuration
    • Full BlockDX setup
    • Blocknet wallet setup
    Using the .zip download (options b/c)
    1. Find the downloaded file. The default location is in Downloads.

    2. Right-click the file, select Extract All.

    3. Select Extract.

    4. After the files are extracted, a new folder should open with the contents. If a folder did not open, look for the new folder in the directory the downloaded file is located.

    5. Here you will find the BLOCK DX executable file.

    6. Double-click the BLOCK DX file to begin the installation process.

    7. Read the License Agreement and select Accept to acknowledge.

    Installation Complete!

    Continue to related guides:

    • BlockDX Configuration
    • Full BlockDX setup
    • Blocknet wallet setup
    MacOS

    1. Download the latest Block DX release. There are 2 MacOS options:

      1. BLOCK-DX-x.x.x-mac.dmg (recommended)
      2. BLOCK-DX-x.x.x-mac.zip

    2. Before continuing, it is recommended to verify your download

    3. Continue to the instructions below for the download you selected:
    Using the .dmg download (option a)
    1. Find the downloaded file. The default location is in Downloads.
    2. Double-click the file to begin installation.

    3. Click and drag the BLOCK DX application icon over to the Applications folder and release.

    4. If you are prompted with a message asking if you would like to replace an existing version with a newer version, select Replace.

    5. Open Finder, navigate to Applications, and find Blocknet in the list of applications.

    6. Right-click the file and select Open. If using the touch pad this can be done by clicking with 2 fingers.

    7. If you are prompted with a message asking if you are sure you want to open the application, select Open.

      Note: If you see a message like the one below instead of the one above, click OK and repeat step 6 above.

    8. Your computer will begin verifying the application. This may take a few minutes to complete.

    9. Read the License Agreement and select Accept to acknowledge.

    Installation Complete!

    Continue to related guides:

    • BlockDX Configuration
    • Full BlockDX setup
    • Blocknet wallet setup
    Using the .zip download (option b)
    1. Find the downloaded file. The default location is in Downloads.
    2. Double-click the file to unpack the contents.
    3. Click and drag the BLOCK DX application over to the Applications folder and release. This will add it to Applications.

    4. If you are prompted with a message asking if you would like to replace an existing version with a newer version, select Replace.

    5. The downloaded file may now be removed.

    6. In Finder, navigate to Applications, and find BLOCK-DX in the list of applications.

    7. Right-click the file and select Open. If using the touch pad this can be done by clicking with 2 fingers.

    8. If you are prompted with a message asking if you are sure you want to open the application, select Open.

    9. Your computer will begin verifying the application. This may take a few minutes to complete.

    10. Read the License Agreement and select Accept to acknowledge.

    Installation Complete!

    Continue to related guides:

    • Blocknet wallet setup
    Linux
    1. Download the latest Block DX release. There are 3 Linux options:
      1. BLOCK-DX-x.x.x-linux.deb (recommended)
      2. BLOCK-DX-x.x.x-linux.tar.gz (Gzip archive)
      3. BLOCK-DX-x.x.x-linux-x86_64.AppImage (For Fedora Linux)
    2. Before continuing, it is recommended to verify your download
    3. Continue to the instructions below for the download you selected:
    Using the .deb download (option a)

    1. You may be asked for a confirmation to download, select Save File then OK.

    2. Find the downloaded file. The default location is in Downloads.

    3. Double-click the file to open the installation screen.

    4. Select Install.

    5. You may be asked to enter your account password.

    6. The installation process may take a few minutes.

    7. Read the License Agreement and select Accept to acknowledge.

    Installation Complete!

    Continue to related guides:

    • BlockDX Configuration
    • Full BlockDX setup
    • Blocknet wallet setup
    Using the .tar.gz download (option b)

    1. You may be asked for a confirmation to download, select Save File then OK.

    2. Find the downloaded file. The default location is in Downloads.

    3. Right-click the file, select Extract Here.

    4. Double-click the BLOCK-DX-x.x.x-linux/ folder to view the contents.

    5. Here you will find the block-dx executable file.

    6. Double-click the block-dx file to install.

    7. Read the License Agreement and select Accept to acknowledge.

    Installation Complete!

    Continue to related guides:

    • BlockDX Configuration
    • Full BlockDX setup
    • Blocknet wallet setup
    Using the .AppImage download (option c)
    1. Save the .AppImage file wherever you normally keep your .AppImage files.
    2. In the directory where the .AppImage file is saved, launch BlockDX as follows: 
      ./BLOCK-DX-x.x.x-linux-x86_64.AppImage --disable-gpu-sandbox
      where x.x.x is the version you downloaded. Note, if that doesn't work, you might also try passing this param: 
      --disable-seccomp-filter-sandbox
    Installation Complete!

    Verifying Downloads¶

    It is important to verify the integrity of downloads before running them. Depending on how you downloaded it, it's possible the file may have been modified in transit to do something evil when run. The server hosting the download may also have been compromised.

    1. Get the sha256 hash of the release you download. These are provided on the Github release page as plain text in the release notes. The format follows SHA256-hash <filename>. Here is an example of the hashes: 
      4a6bd1ad96edf6b052e9929633f978a9a06c85552091a5df1e9bf0bda18ee449  BLOCK-DX-1.9.0-linux-amd64.deb
d90ad1ff0797fc30d749597252718cb7c52843e1020818edb2fbb2a8dbaf95d0  BLOCK-DX-1.9.0-linux-armv7l.AppImage
9fdda89ef175621e052b6ff8dbb6ab0826c6bc85e1f380c0ea967f638577373b  BLOCK-DX-1.9.0-linux-x64.tar.gz
2d23161771b6bcec2a6c6242e872638ac7c6be7864f4bb74df4d7af723716a94  BLOCK-DX-1.9.0-linux-x86_64.AppImage
83e07323a04873a96264f6f46f541956adfa0f86d17986d6278f773660e704db  BLOCK-DX-1.9.0-mac.dmg
6e8211d56204acd155273c19329eea629ff5fc6195e178e026734cc3b3ce354f  BLOCK-DX-1.9.0-mac.zip
45b239611cd4e248d291cd33a9ac7dede927ed4313b29d3eb5728c38420066f2  BLOCK-DX-1.9.0-win-ia32.zip
cc29caeae49b6f99e28b36f64e678267be46a14526fbef4a311acc215aaa3c79  BLOCK-DX-1.9.0-win-x64.zip
8be6d4bde7fa182d68047524671ac61b3366c9b3e8f241bf2493e001fa3502ef  BLOCK-DX-1.9.0-win.exe
    2. Take a note of the hash for the specific file you downloaded.

    3. Get the sha256 hash of the file you downloaded:

      Windows
      1. Open the command prompt.
      2. Navigate to the location of the downloaded file.
      3. Enter certUtil -hashfile filename SHA256 with filename replaced by the name fo the file you downloaded.

      Example: certUtil -hashfile BLOCK-DX-1.9.0-win.exe SHA256

      MacOS
      1. Open the terminal.
      2. Navigate to the location of the downloaded file.
      3. Enter shasum -a 256 filename with filename replaced by the name fo the file you downloaded.

      Example: shasum -a 256 BLOCK-DX-1.9.0-mac.dmg

      Linux
      1. Open the terminal.
      2. Navigate to the location of the downloaded file.
      3. Enter sha256sum filename with filename replaced by the name fo the file you downloaded.

      Example: sha256sum BLOCK-DX-x.x.x-linux.deb

    4. Compare the release hash to the hash of the download. If the hashes do not match, DO NOT run the file and delete the file immediately.
