    Lock/Unlock Wallet¶

    Controlling when the wallet is locked, fully unlocked, or unlocked just for staking is important for increasing the security of your funds. You must first encrypt the wallet to be able to lock it.

    • Locked - When the wallet is locked, you are unable to spend funds or to stake. However, you can still navigate around it, see your balance and transaction history, receive funds, create an address, and edit address labels. If somebody were to gain access to your computer, they would not be able to access your funds without your passphrase to unlock it.
    • Unlocked - When the wallet is unlocked, all functionality of the wallet is available, including spending your funds and staking. It is recommended to only have your wallet fully unlocked when you are spending funds and making transactions. If somebody were to gain access to your computer, they could easily access and steal your funds.
    • Unlocked for staking only - When the wallet is unlocked for staking, it has the security of the wallet being locked, but has the benefit of allowing your funds to stake.

    The passphrase used to encrypt the wallet is required to fully unlock the wallet or just unlock for staking.

    View Lock/Unlock State¶

    View state using the redesigned wallet

    1. In the upper-right corner of the wallet you will see a set of icons.

    2. The blue padlock icon on the right represents the lock/unlocked state.

    3. Hover over the icon to read the current state.

    View state using the classic wallet

    1. At the bottom-right corner of the wallet you will see a set of icons.

    2. The padlock icon on the left represents the lock/unlocked state.

    3. Hover over the icon to read the current state.

    Lock Wallet¶

    Lock using the redesigned wallet

    There are 2 ways to lock the wallet in the redesign:

    Lock using status icon
    1. Open the wallet.

    2. In the upper-right corner of the wallet you will see a set of icons.

    3. The blue padlock icon on the right can be used to change the locked state.

    4. Click on the icon.

    5. Select Lock Wallet.

    6. The wallet is now locked.
    Lock using the program menu
    1. Open the wallet.

    2. In the program menu, select Settings then Lock Wallet.

    3. The wallet is now locked.

    Lock using the classic wallet

    1. Open the wallet.

    2. In the program menu, select Settings then Lock Wallet.

    3. The wallet is now locked.

    Unlock for Staking¶

    Unlock for staking using the redesigned wallet

    There are 2 ways to unlock the wallet for staking in the redesign:

    Unlock for staking using status icon
    1. Open the wallet.

    2. In the upper-right corner of the wallet you will see a set of icons.

    3. The blue padlock icon on the right can be used to change the locked state.

    4. Click on the icon.

    5. Select Unlock Wallet for Staking Only.

    6. A window will appear asking for passphrase.

    7. Enter your passphrase, select OK.

    8. The wallet is now unlocked for staking only.
    Unlock for staking using the program menu
    1. Open the wallet.

    2. In the program menu, select Settings then Unock Wallet.

    3. A window will appear asking for passphrase.

    4. Ensure the Unlock for staking only option is selected.

    5. Enter your passphrase, select OK.
    6. The wallet is now unlocked for staking only.
    Unlock for staking using the classic wallet

    1. Open the wallet.

    2. In the program menu, select Settings then Unlock Wallet.

    3. A window will appear asking for passphrase.

    4. Ensure the Unlock for staking only option is selected.

    5. Enter your passphrase, select OK.
    6. The wallet is now unlocked for staking only.

    Fully Unlock¶

    Unlock using the redesigned wallet

    There are 2 ways to fully unlock the wallet in the redesign:

    Fully unlock using status icon
    1. Open the wallet.

    2. In the upper-right corner of the wallet you will see a set of icons.

    3. The blue padlock icon on the right can be used to change the locked state.

    4. Click on the icon.

    5. Select Unlock Wallet.

    6. A window will appear asking for passphrase.

    7. Enter your passphrase, select OK.

    8. The wallet is now fully unlocked.
    Fully unlock using the program menu
    1. Open the wallet.

    2. In the program menu, select Settings then Unlock Wallet.

    3. A window will appear asking for passphrase.

    4. Ensure the Unlock for staking only option is not selected.

    5. Enter your passphrase, select OK.
    6. The wallet is now fully unlocked.
    Unlock using the classic wallet

    1. Open the wallet.

    2. In the program menu, select Settings then Unlock Wallet.

    3. A window will appear asking for passphrase.

    4. Ensure the Unlock for staking only option is not selected.

    5. Enter your passphrase, select OK.
    6. The wallet is now fully unlocked.
